# 🚀 PrepBuddy - AI Powered Interview Preparation Platform

PrepBuddy is an AI-powered interview preparation platform designed to help students and job seekers practice technical and HR interviews with real-time AI interaction, resume analysis, ATS scoring, and performance feedback.

---

# ✨ Features

## 🤖 AI Mock Interviews

* Technical + HR interview questions
* Dynamic AI-generated questions
* Real-time interaction

## 📄 Resume Analysis

* Upload PDF resumes
* ATS score generation
* Resume-based interview questions

## 🎤 Voice Interaction

* Voice-based interview support
* Speech-to-text answer handling

## 📊 AI Feedback System

* Performance analysis
* Interview feedback
* Improvement suggestions

## ☁️ Cloud Storage

* Resume upload support
* Cloudinary integration
* Resume history management

## 🎨 Premium UI

* Modern dark theme
* Responsive design
* Smooth user experience

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Harman2156/prepbuddy-ai-interview-platform.git
```

---

## Install Frontend Dependencies

```bash
cd client
npm install
```

---

## Install Backend Dependencies

```bash
cd server
npm install
```

---

# 🔑 Environment Variables

Create `.env` file inside `server` folder.

```env
MONGO_URI=your_mongodb_uri

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret

PORT=5000
```

---

# ▶️ Run Project

## Start Backend

```bash
cd server
npm start
```

## Start Frontend

```bash
cd client
npm run dev
```
